# System & User Artifacts

This section covers both system-level and user-level artifacts, which together form the foundation of most forensic investigations.
System artifacts are generated by the operating system and typically reflect system-wide activity. These artifacts are often OS-specific in both format and location. For example, both Windows and Linux maintain logs, but they differ significantly in structure, storage paths, and tooling used for analysis.


User artifacts, on the other hand, are generated through user activity and applications. These tend to be more consistent across operating systems. For instance, a browser like Firefox stores similar types of data (history, cookies, downloads) regardless of whether it is running on Windows or Linux—though the file locations and paths may differ.
